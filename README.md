## ğŸ¨ Customization Guide

### 1. Mengubah Tema Default

Edit `components/ThemeProvider.tsx`:

```tsx
const [accentColor, setAccentColor] = useState<AccentColor>("blue"); // Ubah warna default
const [grayColor, setGrayColor] = useState<GrayColor>("mauve");      // Ubah gray default
```

### 2. Menambah Custom Font

1. Tambahkan font ke `public/fonts/`
2. Daftarkan di `app/globals.css`:

```css
@font-face {
  font-family: "YourFont";
  src: url("/fonts/YourFont.woff2") format("woff2");
  font-weight: normal;
  font-style: normal;
}

.font-your-font {
  font-family: "YourFont", sans-serif;
}
```

### 3. Menambah Lottie Animation

1. Export animation dari [LottieFiles](https://lottiefiles.com/)
2. Simpan JSON ke `public/emoticons/`
3. Gunakan di component:

```tsx
import Lottie from "lottie-react";
import animationData from "@/public/emoticons/your-animation.json";

<Lottie animationData={animationData} loop={true} />
```

### 4. Responsive Breakpoints

Gunakan utility classes di `globals.css`:

```tsx
<div className="res-col res-col-tablet res-col-desktop-6">
  {/* Mobile: 12 cols, Tablet: 6 cols, Desktop: 6 cols */}
</div>
```
